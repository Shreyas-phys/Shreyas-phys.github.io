Machine Learning Projects

  • Spatiotemporal Density: Linear Regression from scratch using the normal equation

    This project applies linear regression from first principles to model the light-cone-like spread of correlations in a quantum spin system. Using data generated from quantum dynamics simulations, I identify the propagation front by fitting a linear model via the normal equation. This forms the foundation for exploring more advanced techniques like polynomial regression, cross-validation, and regularization to capture nonlinear spreading behavior in future work.
