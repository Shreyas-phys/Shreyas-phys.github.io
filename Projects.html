Spatiotemporal Density: Linear Regression from Scratch

This project demonstrates the use of linear regression to fit the first arrival points in a spatiotemporal density map, using the normal equation:

The Normal Equation:

θ = (Xᵀ X)⁻¹ Xᵀ y

Where:

  • θ is the parameter vector (intercept and slope)
  • X is the design matrix, consisting of a column of ones and the input x-values (site indices)
  • y is the vector of observed first arrival times
